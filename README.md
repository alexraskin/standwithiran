# Stand With Iran ✊

[![Stand With Iran](https://i.gyazo.com/13dac932e9348a60e41b49844cc1b274.png)](https://gyazo.com/13dac932e9348a60e41b49844cc1b274)

Welcome to the Stand With Iran project! This website is designed to help share resources supporting the people of Iran in their fight for freedom and human rights. Use it to spread awareness about fundraisers, demonstrations, and organizations working to support the Iranian people during this critical time.

**Woman, Life, Freedom — زن، زندگی، آزادی**

## Docker

```bash
./start.sh
```

or with migrations:

```bash
./start.sh --migrate
```
